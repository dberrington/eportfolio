Types of errors

Syntax Error

a syntax error is an error in the syntax of a sequence of characters or tokens that is intended to be written in a particular programming language. A program will not compile until all syntax errors are corrected.

Runtime Error

A runtime error is a program error that occurs while the program is running. The term is often used in contrast to other types of program errors, such as syntax errors and compile time errors. There are many different types of runtime errors. One example is a logic error, which produces the wrong output.

Logic Error

A logic error is a bug in a program that causes it to operate incorrectly, but not to terminate abnormally (or crash).
