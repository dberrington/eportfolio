String Manipulation

There are four types of data, string, integer, float and boolean.


String

String is used for text and things like that eg "My apple is red" or a phone number or postcode. You collect this data using quotation marks.


Integer

Integers are used for whole numbers like 9, 35 and 2345. You collect this data using int(input(


Float

Float is the same as integer but for decimals, eg 456.57 and 2345.4982. You collect this data using float(input(.


Boolean

Boolean is used for something with two options, eg on and off, yes or no or true or false. You collect this data using bool(input(.


Arrays

An array is a data structure that groups all related data under one identifier (name). In the example above we have used three identifiers (firstNameOne, firstNameTwo and firstNameThree). A static array is a fixed size which means when you create the array you must specify the number of items it will hold and if you want to store more data it will not let you as it will be full, the size of a static array must be stated when you create it in your code eg Friends [5]. A dynamic array will adjust to let you add more items if required.


Index

The index is the number of the item. If you wanted to print the third item in the array you could write print(Friends[3])


Tuple

Tuples use these brackets (), you can't change the values (immutable). Each value is numbered starting from zero, for easy reference.


Dictionary

Dictionaries use these brackets {}, you can change the values (mutable). In a dictionary, you have an 'index' of words, and for each of them a definition. In python, the word is called a 'key', and the definition a 'value'. The values in a dictionary aren't numbered - tare similar to what their name suggests - a dictionary. In a dictionary, you have an 'index' of words, and for each of them a definition. In python, the word is called a 'key', and the definition a 'value'. The values in a dictionary aren't numbered - they aren't in any specific order, either - the key does the same thing. You can add, remove, and modify the values in dictionaries. Example: telephone book.


List

Tuples use these brackets [], you can change the values (mutable). Each one of them is numbered, starting from zero - the first one is numbered zero, the second 1, the third 2, etc. You can remove values from the list, and add new values to the end. Example: Your many cats' names.
